
# Model Weights

The trained model weights are not included in this repository due to GitHub file size limitations.

## Model Information

* Model: YOLOv8
* Task: Pavement Distress Detection
* Framework: Ultralytics YOLOv8
* Classes:

  * Longitudinal Crack
  * Transverse Crack
  * Alligator Crack
  * Repaired Crack
  * Pothole
  * Patching
  * Rutting

## Obtaining Weights

If you require the trained model weights for academic or research purposes, please contact the repository author.

After obtaining the weights, place the file in:

```text
models/
└── best.pt
```

## Notes

The repository contains the complete implementation, training pipeline, web application, PCI estimation module, and documentation required to reproduce the project.
